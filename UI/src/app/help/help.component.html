Stellar-Evolution-Clustering | How to Use the Application

Query Form

1. Any clustering begins by filling out the query form

2. The first item on the form is the selection of attributes

3. Select the attributes you'd wish to cluster by, and add them to the right hand side

4. Next, define weights for each attribute. The larger the value, the more that attribute will effect clustering. The weights should add to 1.00

5. Select the cluster method you'd like to use. The two clustering methods provided are K-means and DBScan

6. Finally, review the clustering form and submit to visualize the data
